自動クリック

概要

PyAutoGuiを使って１分毎にクリックを発生させるもの。 人が居なくても、PCの画面のある位置を１分毎にマウスでクリックする操作を自動で発生させるもの。
Windows向け。pythonを利用。

内容

click_gamen1.py

１分毎に、画面の(10,10)の位置にクリックを発生させる。
画面の位置ix,iyは、PCのアプリが動作しないような、支障のない位置に設定する。

click_gamen1.py

画面からcmd.pngの部分を探して、そこをクリックする。
cmd.pngに相当する画面がないと、エラーが発生する。稀に、上手く動作しないこともある。
